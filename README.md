# MealMate

A responsive food-delivery website built with HTML and CSS. Provides restaurant browsing, onboarding, and real-time order tracking.

## Table of Contents
- Features
- Pages
- How to run
- Contact
- Planned upgrades

## Features
- Browse local restaurants and favorite dishes
- Onboarding flow for meal and restaurant selection
- Order tracking with progress bar and estimated delivery time
- Responsive layouts for mobile and desktop

## Pages
- `index.html` — Homepage: hero, "how it works", featured restaurants, testimonials  
- `onboarding.html` — Onboarding screens to select favorite dishes and restaurants  
- `tracking.html` — Order tracking page with progress bar and map placeholder  

## How to run
1. Open `index.html` in a web browser.  
2. (Optional) Serve the folder with a static server (e.g., VS Code Live Server or `npx http-server`) for consistent behavior.
